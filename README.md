# 🏆 Simulador Predictivo IA - Mundial de Fútbol 2026

Simulador avanzado de torneo de fútbol que utiliza el **Modelo Econométrico de Joachim Klement** (PIB, población, pedigrí y localía) para predecir resultados y el campeón del mundo.

## ✨ Características

- 🔮 Simulación completa de fase de grupos (12 grupos)
- 📊 Tabla de mejores terceros (8 clasificados)
- 🏆 Fase de eliminación directa completa (R32, R16, Cuartos, Semifinales, Final)
- 🤖 Algoritmo de IA con factores económicos y de rendimiento
- 🎲 Outsider (Caballo Negro) aleatorio por simulación
- 📱 Conexión a WhatsApp para adquisición de acceso ($1.00 USD)
- 👥 Sistema de usuarios con roles (Admin/Cliente)
- 🖨️ Exportación a PDF optimizada
- 🎨 Interfaz moderna con Tailwind CSS
